Projects

Dancing Links Sudoku Solver

A program made to solve Sudoku puzzles using the Dancing Links algorithm. Can solve puzzles up to size 25 by 25 regardless of the amount of numbers placed. Made in Python. Link to the program is here.

Text Adventure

A text-adventure based program that is controlled via keystrokes (i.e. WASD for movement, E to interact, etc.) and shows an ASCII based map as a display. It currently shows a single room that the player can escape from. Made in Python using the PyCharm IDE. Link to the game is here.

Firearm Simulator

A text-based program that simulates various firearms and uses various keystrokes to operate the firearms. It currently simulates break action (single and multiple barrel firearms with single triggers) and single shot firearms (break action and bolt action), as well as simulating ammunition malfunctions and operations specific to certain firearms. Made in Python using the PyCharm IDE. Link to the application is here.
